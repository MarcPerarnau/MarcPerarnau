# 💻 **Marc Perarnau** | Ciberseguridad & Full-Stack Developer  

## 🚀 Sobre mí  
Soy un **estudiante en ciberseguridad y desarrollo full-stack** con experiencia en la implementación de infraestructuras seguras, gestión de redes y desarrollo de aplicaciones web. Mi enfoque combina seguridad ofensiva y defensiva, administración de sistemas y programación para crear soluciones robustas y escalables.  

📍 Actualmente, estoy desarrollando un **sistema de detección y eliminación de botnets** en redes empresariales, integrando IPv6, Proxmox y seguridad activa.
